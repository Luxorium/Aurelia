# Code Of Conduct

Aurelia aims to be a professional, contributor-friendly open-source project.

## Expected Behavior

- Be respectful and direct.
- Assume good intent, but be willing to clarify technical claims with evidence.
- Keep feedback focused on the work, not the person.
- Respect the clean-room policy and the legal boundaries of the project.
- Make space for contributors with different levels of Rust, networking, and Minecraft protocol experience.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Posting private information without consent.
- Deliberately derailing technical discussions.
- Pressuring contributors to include legally unsafe material.
- Submitting Mojang source code, Minecraft assets, generated jars, decompiled source, or copied server/modding project code.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests; request changes; or block participants when behavior harms the project or its contributors.

If you need to report a conduct issue, contact the maintainers through the most appropriate private GitHub channel available for the repository. If no private channel is enabled, open a minimal public issue asking for maintainer contact without sharing sensitive details.
