# Aurelia 0.2.0 - Vanilla Parity Foundation

The first public `v0.2.0` GitHub release.

## Highlights

- Experimental real-client flat-world join path for a clean Minecraft Beta 1.7.3 client.
- Basic movement tracking and chunk streaming.
- Chunk load/unload visibility updates when crossing chunk boundaries.
- Conservative inventory sync and click handling for the player inventory window.
- Starter hotbar and inventory-backed block break/place testing.
- Complete clean-room Beta 1.7.3 rule tables: every block id (`0..=96`) and item id (`256..=357` plus both music discs) with Beta-era stack sizes, hardness, tool/tier harvest requirements, drops, and light emission. Public-documentation values are isolated and marked approximate pending trace verification.
- Technical block forms (fluids, fire, door/sign/bed/repeater blocks, piston internals, portal) are never placeable from inventory stacks, including from hostile clients.
- Hardened protocol and persistence paths: bounded chat string reads, collision-free player save file names, and single-lock login state handling.
- Chat/debug commands for local testing.
- Aurelia-native dirty flat-world edit persistence.
- Aurelia-native basic player-state persistence and clean rejoin with saved edits.

## Known Missing Systems

- Verified production login response semantics.
- Crafting.
- Chests.
- Furnaces and smelting.
- Workbench UI.
- Item entities, pickups, and overflow drops.
- Mobs, AI, visible entity packets, and combat.
- Vanilla terrain generation.
- Vanilla McRegion/NBT world and player persistence parity.

## Downloads

A prebuilt Linux x86_64 `aurelia-server` binary is attached to the GitHub release. On other platforms, build from source:

```bash
cargo build --release -p aurelia-server
```

## Compatibility Notes

This release is an early compatibility foundation, not a complete Beta 1.7.3 server. The real-client path is experimental and should be tested with:

```bash
cargo run -p aurelia-server -- --host 127.0.0.1 --port 25565 --experimental-join --playable-flat-world --chunk-radius 1 --compat-debug --trace-packets --trace-packet-limit 512
```

Current persistence is Aurelia-native and not vanilla McRegion/NBT-compatible.
