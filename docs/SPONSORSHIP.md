# Sponsorship

Aurelia is an early open-source compatibility project. Sponsorship is optional, but it helps keep the work steady and public.

## What Sponsorship Supports

Sponsorship helps with:

- Clean-room protocol research.
- Real-client compatibility testing.
- Documentation and contributor onboarding.
- Hosting and build infrastructure.
- Long-term development time for world, inventory, persistence, entity, and region-threading work.

## Project Boundaries

Sponsorship does not change the clean-room policy. Aurelia will not accept Mojang source code, Minecraft assets, generated jars, decompiled source, copied protocol code, or copied server/modding project implementations.

Sponsorship also does not imply a guarantee that a specific feature will be implemented on a fixed timeline. The project will continue to prioritize correctness, legal safety, and maintainable architecture.

## GitHub Sponsors

The repository is configured for GitHub Sponsors with:

```yaml
github: Luxorium
```

The maintainer still needs to verify GitHub Sponsors enrollment and enable the sponsorship button in GitHub repository settings before it appears publicly.
